# 🔐 Laboratório de Ataques de Força Bruta com Medusa (Kali Linux)

## 📖 Descrição do Projeto
Este projeto tem como objetivo **simular ataques de força bruta em ambientes vulneráveis controlados**, utilizando o **Kali Linux** e a ferramenta **Medusa**, com foco em **aprendizado, auditoria de segurança e defesa**.

Foram explorados cenários em serviços comuns como:
- FTP
- Aplicações Web (DVWA)
- SMB (password spraying)

⚠️ **Todos os testes foram realizados em ambientes de laboratório isolados**, sem qualquer impacto em sistemas reais.

---

## 🎯 Objetivos
- Compreender ataques de força bruta em diferentes serviços
- Utilizar o Medusa para auditoria de segurança
- Praticar enumeração básica de usuários
- Documentar tecnicamente processos e resultados
- Propor medidas de mitigação
- Publicar documentação técnica no GitHub como portfólio
